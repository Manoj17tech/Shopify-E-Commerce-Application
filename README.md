# 🛒 React Shopping Cart Application

A modern **Shopping Cart Web Application** built using **React, Redux, and React Router**.
This project demonstrates how to build a simplified **e-commerce cart system** with product listing, cart management, and dynamic price calculation.

---

# 📌 Project Overview

This application allows users to browse products, add them to a cart, update quantities, and view the total cost of items in their cart.

The project demonstrates important frontend concepts such as:

* Component-based architecture
* Global state management using Redux
* Dynamic UI updates
* Routing using React Router
* Clean and responsive UI design

It serves as a **basic foundation for building full-scale e-commerce platforms**.

---

# 🚀 Features

### Product Listing

* Displays product image, title, brand, price, and rating
* Clean card-based product layout

### Add to Cart

* Users can add products to the cart
* Cart icon updates with item count

### Cart Management

* View all items added to the cart
* Update product quantity
* Remove items from cart

### Dynamic Price Calculation

* Calculates total price based on quantity
* Updates subtotal and total automatically

### Responsive UI

* Clean modern layout
* Works across different screen sizes

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/react-shopping-cart.git
```

### 2️⃣ Navigate to project folder

```bash
cd react-shopping-cart
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```

# 🖥️ Application Flow

1️⃣ User visits the home page
2️⃣ Product list is displayed
3️⃣ User clicks **Add to Cart**
4️⃣ Cart icon updates with item count
5️⃣ User navigates to the cart page
6️⃣ User updates quantity or removes items
